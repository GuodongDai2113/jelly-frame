# Jelly Frame

- 主题适用于企业官网，因为我的客户们大概都是企业。
- 专为 B2B 设计，移除了价格，销售，评价等显示
- 如果你的网站使用Hello Elementor主题，大概率是完全兼容的。

## 特点

- 针对Elementor、Woocommerce做了很多调整
